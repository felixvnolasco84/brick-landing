# Brick landing page decisions

- Use the six supplied screenshots as the visual source of truth for hierarchy, spacing, rounded cards, and restrained editorial styling.
- The implementation is mobile-first and must remain usable from 320px upward.
- Brand tokens: ink `#0B0A08`, muted text `#696765`, accent `#E7F256`, soft background `#F4F2F0`.
- Product screenshots, charts, and lifestyle photography are intentionally represented by labeled image slots so the user can add final assets later.
- Prefer relevant shadcn-style primitives over one-off UI controls.
